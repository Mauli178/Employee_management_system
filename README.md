Employee Management System

## Description

The **Employee Management System** is a desktop application developed using **Python**, **Tkinter**, and **MySQL**. It provides a simple graphical interface to manage employee records efficiently. Users can add new employees, remove existing employees, promote employees by updating their salary, and view all employee details. The application automatically creates the required database and table if they do not already exist, making it easy to set up and use.

## Features

- Add Employee
- Remove Employee
- Promote Employee
- Display Employee Records
- Automatic Database and Table Creation

## Technologies Used

- Python
- Tkinter
- MySQL
- mysql-connector-python
